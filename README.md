# Nombre del proyecto: Portafolio

Portafolio personal, para presentar mis habilidades, proyectos y trayectoria de manera profecional y responsiva.

## Funcionalidad

El sitio es una landing page estática que muestra mi perfil profecional. Esta optimizada para diferentes resoluciones, asegurnado que el contenido se adapte correctamente a dispositivos moviles y tablets.

## Tecnologías y Técnicas utilizadas

- HTML5: Estructura semántica del contenido
- CSS3: Estilos, Flexbox y Grid para el diseño.
- Metodología BEM: Organización de clases para un código escalable y facil de leer.
- Diseño Responsivo: Uso de valores relativos y Media Queries.

- Git y GitHub: Control de versiones y despliegue.
